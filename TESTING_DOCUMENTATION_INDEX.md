## 🔧 Quick Command Reference

```bash
# View all documents
ls -la *.md              # All markdown documents
ls -la *.html            # HTML reports
ls -la *.csv             # CSV data

# Run tests
npm test                 # Unit + integration (149 tests, ~10s)
npm run test:watch      # Watch mode
npm run test:coverage   # Generate coverage report
npm run test:e2e        # E2E tests (80 tests, ~6-8s)
npm run test:e2e:ui     # E2E with interactive UI
npm run test:e2e:headed # E2E with visible browser
npm run test:all        # All tests together (~18s)

# View reports
open TESTING_REPORT.html                    # Browser
open TESTING_REPORT.csv                     # Excel
cat TESTING_QUICK_REFERENCE.md             # Terminal
less END_TO_END_TESTING_SUMMARY.md          # Pager

# Debug
npm test -- --verbose                       # Verbose output
npm test -- --listTests                     # List test files
npm test -- __tests__/lib/encryption.test.ts # Specific test
npm test -- --testNamePattern="encryption"   # Pattern match
```
